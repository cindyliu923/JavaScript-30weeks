# JavaScript-30weeks
命運乖舛的 Repo Q

## Rules 
1. 參與請開 Branch (沒有命名規，你開心就好)
2. PR 請自己 Merge 
3. 有問題就先自己想辦法，想破頭想不出來請推上來之後，去賴群組發問
4. 暫時還沒想到第四條

## ルール
1. このチャレンジを参加したい方は自分でブランチを作成してください。（ブランチの名前を自分好きにしてください。）
2. PR は自分でマージしてください。
3. 問題があったら、自分で先に解決方法を探してください。どうしても分からなかったら、ライングループに問題を提出してください。
4. 四つ目暫く思いつかない
